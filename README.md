# Engineering Books

A collection of engineering books generated from company blogs.

## Available Books

### [Anthropic Engineering](anthropic-engineering/)
A compilation of articles from the [Anthropic Engineering Blog](https://www.anthropic.com/engineering).

- **PDF**: [anthropic_engineering_book.pdf](anthropic-engineering/anthropic_engineering_book.pdf)
- **Source**: [generate_book.py](anthropic-engineering/generate_book.py)

## Usage

To generate a book yourself:

1.  Navigate to the specific book directory (e.g., `anthropic-engineering`).
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the generation script:
    ```bash
    python generate_book.py
    ```
